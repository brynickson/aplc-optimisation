Apodized Pupil Lyot Coronagraph design optimization
====================================================

`aplc-optimization` is an object-orientated software toolkit to automate the exploration of APLC design parameter combinations.

Documentation can be found online at https://aplc-optimization.readthedocs.io/

Requirements & Installation
---------------------------

***Prerequisites:***

 * numpy, scipy, matplotlib, etc.
 * astropy
 * pysynphot
 * poppy (>= version 0.7.0)
